# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-05-02

### Added
- ✨ Initial release of Starbucks Egypt React Clone
- 🎨 Complete UI implementation with React 18 + TypeScript
- 🎯 Sticky navigation bar with mobile menu
- 🖼️ Hero banner with Gaza donation message
- 📝 Statement section with official content
- 🎴 Featured cards grid with 5 promotional cards
- 🦶 Footer with multi-column links and country selector
- 🎭 Framer Motion animations throughout
- 🎨 Tailwind CSS v4 with custom Starbucks theme
- 🧩 shadcn/ui components (Button, Card, Sheet)
- 🌐 Full RTL (Right-to-Left) support for Arabic
- 📱 Fully responsive design (mobile-first)
- 🔤 Cairo font integration from Google Fonts
- 🐳 Docker and Docker Compose configuration
- 🔄 GitHub Actions CI/CD workflows
- 📚 Comprehensive documentation in `docs/` folder
- 🔒 Security scanning with CodeQL
- 🤖 Dependabot configuration for automated updates
- 📄 MIT License
- 🤝 Contributing guidelines
- 🐛 Issue templates for bugs and features
- 📋 Pull request template

### Technical Stack
- React 19.2.5
- TypeScript 6.0.2
- Vite 8.0.10
- Tailwind CSS 4.2.4
- Framer Motion 12.38.0
- Lucide React 1.14.0
- shadcn/ui components
