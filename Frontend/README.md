## 📖 About

**Starbucks Egypt** is a production-grade React frontend clone built to demonstrate enterprise-scale engineering. It features a complete **bilingual (Arabic/English) experience** with full RTL support, a data-driven component architecture, and modern deployment patterns.

### Key Highlights
- 🌍 **Bilingual UX:** Seamless switching between Arabic (RTL) and English (LTR).
- 📱 **PWA Ready:** Installable, offline-ready Progressive Web App.
- ⚡ **Performance:** Powered by React 19, Vite 8, and TanStack Query v5.
- 🛡️ **CI/CD:** Automated security audits, linting, and multi-stage Docker builds.

> This project is an **educational clone**. It is not affiliated with or endorsed by Starbucks Corporation.

## 🚀 Quick Start

```bash
git clone https://github.com/Mostafa-SAID7/starbucks.git
cd starbucks
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) — done! ☕
