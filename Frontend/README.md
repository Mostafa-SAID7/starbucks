# ☕ Starbucks Egypt — Frontend

The React-based presentation layer of the Starbucks Egypt enterprise application.

---

## 🏗️ Overview

This directory contains the **React 19** frontend, built with **Vite**, **TypeScript**, and **Tailwind CSS v4**. It communicates with the .NET backend to provide a seamless coffee commerce experience.

---

## 🚀 Getting Started

To run the frontend locally:

```bash
# Install dependencies
npm install --legacy-peer-deps

# Start development server
npm run dev
```
