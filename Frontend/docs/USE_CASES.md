# 🎯 Project Use Cases

Primary workflows and user goals supported by the application.

## 🌍 Bilingual Product Discovery
**User Goal:** Browse the Starbucks menu in a preferred language (Arabic or English).
- **Workflow:** Switch language via the Navbar toggle.
- **Outcome:** Instant layout flip (RTL/LTR) and content translation for all items, categories, and descriptions.

## 📍 Store Finder (Locations)
**User Goal:** Find the nearest Starbucks store in a specific city.
- **Workflow:** Navigate to `/locations`, filter by city (e.g., Cairo, Alexandria).
- **Outcome:** Filtered list of stores with operating hours and address details.

## 📱 Mobile PWA Installation
**User Goal:** Access the Starbucks menu quickly from the home screen.
- **Workflow:** Select "Add to Home Screen" in a mobile browser.
- **Outcome:** Application installed as a standalone PWA with offline caching.

## 🌟 Rewards Exploration
**User Goal:** Understand the benefits of the loyalty program.
- **Workflow:** Visit the `/rewards` page to view tiers and star-earning rules.
- **Outcome:** Informed user ready to join the loyalty program.
