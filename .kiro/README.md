## 🚀 Quick Start

```bash
# Setup
cd Frontend
npm install

# Development
npm run dev

# Testing
npm run test              # Unit tests
npm run test:coverage     # Coverage report
npx playwright test       # E2E tests

# Build
npm run build             # Production build
npm run type-check        # TypeScript check
```
